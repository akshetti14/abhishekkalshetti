E-Commerce Website

This project was developed a part of Open Source Contribution Course in 5th Semester.

This is a fully functioning e-commerce platform,flexible for launching any type and any number of products.

The site is also provided with an administration side where you can see the orders made,product quantity left,reports. Admin can add or delete product products very easily and even categorize them.

The site is integrated with paypal payment gateway,which makes all the transactions very smooth and reliable.

Technologies:
  • - HTML
  • - Javascript
  • - CSS
  • - MySql
  • - PHP
